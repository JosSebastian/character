# PyTorch Character Recognition Model (A-Z & 0-9) using CNN

### Requirements:

- Python
- Jupyter
- PyTorch
- NumPy
- Matplotlib

### Dataset:

EMNIST Letters + EMNIST Digits (https://www.nist.gov/itl/products-and-services/emnist-dataset)

### Model:

Convolutional Neural Network (CNN)

### Accuracy:

91%

### Run the Code:

```bash

jupyter run main.ipynb
```
